简介：

    本软件为原神树脂辅助工具。

特色：

    通知栏常驻提醒，通知栏弹窗提醒，桌面自带小程序通知，米游社自动签到，点击桌面小部件自动签到米游社。

    内附cookie获取和米游社签到工具。

教程：

首次使用：

    1、请先点击登录按钮，进行登录。

    2、登录完成之后点击刷新即可使用。

    3、请在系统的通知设置里按需修改所需的通知类型。

    4、请务必关闭省电优化。

    5、请在系统的启动管理里给软件后台运行权限。

    6、在设置页面可设置快捷启动原神的启动类型。

    7、在设置页面可设置桌面小部件字体颜色。

    8、点击小部件进行刷新的时候，只有每日第一次签到会进行通知提醒，后续点击不再进行通知提醒。

常见问题解决：

    1、网盘链接里的zip包是电脑端的，手机不要用。

    2、B服用户在一个米游社账号下有多个原神UID的，可在设置页面选择需要提醒的原神账号UID。

    3、如果显示的全是“读取失败”，一般是点击保存按钮太快的原因，等页面整个加载完成之后再点击保存。

    4、MIUI找不到小部件在哪的话请看群里的教程视频。

    5、如果更新完新版本主界面头像乱了的话，多刷新几次就好了。
